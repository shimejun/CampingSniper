マルチ専用
=============================================
<利用方法>
プレイしたいワールドデータのdatapacksフォルダに、ファイルごと入れてください。
ゲームに入ったら /function owgame:items/get_mods を実行し、入手したアイテムでエリア中心点のランダム決定の中央点を決定してください。

ゲームを開始するときは /function owgame:game を実行してください。
ゲームを実行するプレイヤーには "op" のタグを付けておくと、ゲーム終了時に管理本を入手できます。
もしくは/function owgame:operator/book を実行して入手できます。
