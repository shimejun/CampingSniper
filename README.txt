Minecraft Java Edition 企画用システム 「キャンピングスナイパー」
=============================================
<基本情報>
名称: キャンピングスナイパー
最終更新: 2023/4/22
バージョン: 0.2.1
対応MCバージョン: 1.19.x
作者: しめじゅん
マルチ専用
=============================================
<利用方法>
プレイしたいワールドデータのdatapacksフォルダに入れてください。

<セットアップ>
導入したワールドに入ったら /function owgame:items/get_mods を実行してください。
そして獲得したスポーンエッグを任意の場所に設置してください。(安全地帯の分布の中心になります。)

/function owgame:operator/book を実行して管理本を入手できます。
本の内容をクリックしてゲームを開始したり、詳細な設定ができます。
また、ゲームを実行するプレイヤーには "op" のタグを付けておくと、ゲーム終了時に管理本を自動入手できます。
